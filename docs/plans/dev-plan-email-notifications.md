# Email Notifications ‚Äî Implementation Plan

**Date:** 2026-03-14

---

## Objective

Send a one-time admin email notification when a bootstrap token is generated. The email includes token metadata (site, who generated it, expiry) but never the raw token itself. Support two email providers (AWS SES and SendGrid) with a configuration flag to select the active provider.

## Architecture Decisions

| Area | Decision |
|------|----------|
| Provider selection | Configuration flag (`Provider`: `"AwsSes"` \| `"SendGrid"` \| `"None"`) with `Enabled` toggle |
| Interface location | Application layer (`IEmailNotificationService`) ‚Äî follows `IObservabilityMetrics` pattern |
| Implementations | Infrastructure layer ‚Äî follows `CloudWatchEmfMetricSink` / `FirebasePushHintSender` pattern |
| DI wiring | Factory lambda in `Program.cs` ‚Äî follows `IFieldEncryptor` pattern |
| Failure handling | Best-effort: try/catch in handler, log warning on failure, never fail the token generation request |
| Security | `BootstrapTokenGeneratedEmail` model has no raw token property ‚Äî structurally impossible to leak |
| SendGrid transport | Raw `HttpClient` via `IHttpClientFactory` ‚Äî no extra NuGet SDK dependency |

## Implementation Steps

### Step 1: Interface + Email Model (Application Layer) ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Application/Notifications/IEmailNotificationService.cs`

- `IEmailNotificationService` interface with `SendBootstrapTokenGeneratedAsync` method
- `BootstrapTokenGeneratedEmail` record with: `SiteCode`, `TokenId`, `GeneratedBy`, `ExpiresAt`, `LegalEntityId`
- No raw token in the model

### Step 2: Options Class (Application Layer) ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Application/Notifications/EmailNotificationOptions.cs`

- `Enabled` toggle (default: `false`)
- `Provider` selector (`"AwsSes"` | `"SendGrid"` | `"None"`)
- `FromAddress`, `AdminRecipients[]`
- `AwsSesRegion` (default: `"eu-west-1"`)
- `SendGridApiKey`
- `SectionName = "EmailNotifications"`

### Step 3: No-Op Implementation (Infrastructure Layer) ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Infrastructure/Messaging/NoOpEmailService.cs`

- Logs debug message and returns
- Used when `Enabled = false` or `Provider = "None"`

### Step 4: AWS SES Implementation (Infrastructure Layer) ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Infrastructure/Messaging/AwsSesEmailService.cs`

- Uses `AWSSDK.SimpleEmailV2` NuGet package
- Takes `IOptions<EmailNotificationOptions>` and `ILogger`
- Sends HTML email with token metadata
- Throws on error (handler catches)

### Step 5: SendGrid Implementation (Infrastructure Layer) ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Infrastructure/Messaging/SendGridEmailService.cs`

- Uses raw `HttpClient` to call SendGrid v3 API (`https://api.sendgrid.com/v3/mail/send`)
- Takes `HttpClient`, `IOptions<EmailNotificationOptions>`, `ILogger`
- Same HTML email content as SES
- Throws on error (handler catches)

### Step 6: NuGet Package Reference ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Infrastructure/FccMiddleware.Infrastructure.csproj`

- Added `AWSSDK.SimpleEmailV2` package reference

### Step 7: DI Wiring in Program.cs ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Api/Program.cs`

- `builder.Services.Configure<EmailNotificationOptions>(...)`
- Factory lambda that selects provider based on `options.Provider`:
  - `"AWSSES"` ‚Üí `AwsSesEmailService`
  - `"SENDGRID"` ‚Üí `SendGridEmailService`
  - default / disabled ‚Üí `NoOpEmailService`

### Step 8: Hook into GenerateBootstrapTokenHandler ‚Äî DONE

**File:** `src/cloud/FccMiddleware.Application/Registration/GenerateBootstrapTokenHandler.cs`

- Inject `IEmailNotificationService` as constructor parameter
- After successful save + TOCTOU check, call `SendBootstrapTokenGeneratedAsync`
- Wrapped in try/catch ‚Äî logs warning on failure, never fails the request

### Step 9: Configuration in appsettings ‚Äî DONE

**Files:**
- `appsettings.json` ‚Äî defaults (disabled)
- `appsettings.Development.json` ‚Äî dev config (disabled, with placeholder addresses)

```json
"EmailNotifications": {
    "Enabled": false,
    "Provider": "None",
    "FromAddress": "",
    "AdminRecipients": [],
    "AwsSesRegion": "eu-west-1",
    "SendGridApiKey": ""
}
```

## Email Content

**Subject:** `Bootstrap Token Generated ‚Äî Site {SiteCode}`

**Body includes:**
- Site code
- Token ID
- Generated by (actor display name)
- Expiry date/time
- Note that the raw token is not included for security

**Does NOT include:**
- Raw token value
- QR code
